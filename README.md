# Extension-directe
Une extension pour Google Chrome pour afficher les moyennes dans les différentes matières, ainsi que la moyenne générale.
Cliquez sur l'icône de l'extension pour afficher les moyennes.

![cliquez sur l'icône de l'extension](https://github.com/Hericendre/Extension-directe/blob/main/illustrations/extension%20directe%20illustration.png?raw=true)

## Installation
Allez à l'adresse chrome://extensions/ et activez le mode développeur 

![activez le mode développeur](https://github.com/Hericendre/Extension-directe/blob/main/illustrations/explication%201.png?raw=true)

Cliquez sur charger l'extension non empaquetée et sélectionnez le dossier "extension directe"

![charger l'extension non empaquetée](https://github.com/Hericendre/Extension-directe/blob/main/illustrations/explication%202.png?raw=true)

![sélectionner le dossier](https://github.com/Hericendre/Extension-directe/blob/main/illustrations/explication%203.png?raw=true)

Cliquez sur la pièce de puzzle et épinglez l'extension

![épingler l'extension](https://github.com/Hericendre/Extension-directe/blob/main/illustrations/explication%204.png?raw=true)
